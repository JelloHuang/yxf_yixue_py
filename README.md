python==3.6  
openpyxl  
web.py==0.40.dev0  

python3解决方案：  
安装python3（已整理安装脚本）。  
在项目的所有代码开头加上这两句：  

	#!/usr/bin/python3   
	# -*- coding: utf-8 -*-

## 使用方法1：安装为Python系统模块

install:   

	python3 setup.py install
	
uninstall:   

	pip3 uninstall yxf-yixue
	
## 使用方法2：作为服务运行

runserver:   

	python3 server.py  或者  ./start_server.sh
