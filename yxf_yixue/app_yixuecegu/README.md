yixuececai : 易学测股模块
=========================================================

1.手工从网站复制数据表，将来通过爬虫自动获取数据。  
2.输入程序进行格式化，输出适用于SPSS软件进行回归分析的数据表。  
3.使用SPSS进行回归分析，每个案例都要单独分析，得到回归方程。  
4.手工编辑回归方程数据表。  
5.程序依据回归方程数据表进行预测。
